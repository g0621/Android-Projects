# Load all gallery image Fragment.

### A Fragment that loads all gallery Async on Another Thread.

A base code to implement in your apps.

------



### About:

- Base code to load and show gallery images in a fragment .

- loads images async thats makes UI snappy .

- used default FullScreenActivity as MediaActivity .

  **Note :** Trigger using the camera floating action button.
